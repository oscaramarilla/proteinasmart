## DECISIONES CLAVE POR MÉTRICA

| Si esto pasa... | Entonces haz esto |
|---|---|
| Bounce rate >40% | Cambiar copy/imagen hero, simplificar landing |
| Tasa apertura <20% | Cambiar asunto, testear nuevo horario de envío |
| CTR <2% | Cambiar CTA, color, o copy de botón |
| Conversión landing <5% | Agregar prueba social, testear precio, mejorar UX |
| Retención <80% | Revisar contenido de emails, es muy salesy? |
| NPS <7 | Llamada 1:1 con cliente, entender pain point |
| Pipeline vacío | Aumentar gasto en ads o hacer outreach directo |
